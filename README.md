# SIP App

Privacy-first Web3 application interface for SIP Protocol.

**Live URL:** https://app.sip-protocol.org (coming soon)

## About

This repository contains the web application for SIP Protocol - enabling private transactions across multiple blockchains.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```
